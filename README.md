# Prime-Media Messenger

**Prime-Media Messenger** — корпоративный мессенджер для организаций численностью до **250 сотрудников** с Android-клиентом и собственным сервером на Python.

Проект разработан как независимая система корпоративного обмена сообщениями без использования сторонних облачных сервисов. Поддерживает защищённое соединение по SSL/TLS, обмен текстовыми сообщениями и файлами, локальное хранение данных и разграничение прав пользователей.

---

* Авторизация пользователей

* Отправка текстовых сообщений

* Локальное хранение истории сообщений (Room)

* Material Design интерфейс

* Python 3.8+
* SSL/TLS соединения
* SQLite база данных
* Многопоточная обработка клиентов
* Собственный протокол обмена
* Автоматическая инициализация базы
* Хеширование паролей (SHA-256 + Salt)
* Резервное копирование базы данных

# Установка сервера

## 1. Перейдите в каталог

```bash
cd server
```

## 2. Сгенерируйте сертификаты

```bash
chmod +x generate_certs.sh
./generate_certs.sh
```

Будут созданы

```
certs/cert.pem
certs/key.pem
```

---

## 3. Запустите сервер

```bash
python3 main.py
```

или

```bash
python3 main.py -d
```

для режима отладки.

При первом запуске автоматически создаются:

* база данных
* тестовые пользователи
* необходимые таблицы

# Настройка Android

Перед сборкой необходимо указать IP сервера.

В файле

```
app/src/main/res/values/strings.xml
```

измените

```
server_host
```

например

```
192.168.1.15
```

или

```
10.0.2.2
```

для Android Emulator.

Порт по умолчанию:

```
12345
```

После изменения выполните сборку проекта.
